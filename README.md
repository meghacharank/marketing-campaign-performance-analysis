# Marketing Campaign Performance Analysis

## Objective
The objective of this project is to analyze digital marketing campaign performance across multiple channels and identify opportunities to improve efficiency using key marketing metrics.

## Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Dataset
The dataset includes campaign-level data such as impressions, clicks, conversions, and ad spend across different marketing channels.

## KPIs Analyzed
- Click-Through Rate (CTR)
- Conversion Rate
- Cost Per Click (CPC)
- Cost Per Acquisition (CPA)

## Key Insights
- Google Search generated the highest number of conversions.
- Instagram Ads showed better cost efficiency compared to Facebook Ads.
- Facebook Ads achieved high reach but lower conversion performance.

## Business Recommendations
- Increase budget allocation toward high-performing search campaigns.
- Optimize social media creatives to improve conversion rates.
- Focus more on cost-efficient channels to reduce acquisition costs.
